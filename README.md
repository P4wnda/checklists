# p4wnda Checklists

Static GitHub Pages app for CTF box checklists.

Live URL:

```text
https://p4wnda.github.io/checklists/
```

## Features

- Standalone box checklist
- Active Directory box checklist
- Per-template progress saved in the browser
- Machine name, notes, reset, and Markdown export
- Direct links into the pentest cheatsheet using `/pentest/?q=...`

## Deployment

This repository is intended to be served by GitHub Pages from:

- Branch: `master`
- Folder: `/ (root)`

No build step is required.
